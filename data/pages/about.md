# About eXist-db

eXist-db is an open source native XML database and application platform built on XML technology. It stores data as XML documents and provides XQuery for querying, transforming, and building applications directly on your data.

## History

The eXist-db project was started in 2000 by Wolfgang Meier at TU Darmstadt, Germany. It has since grown into a mature, production-grade database used by organizations worldwide — from university research projects to government agencies to commercial publishers.

## Key Features

- **Native XML storage** — documents stored in their natural hierarchical form
- **XQuery 3.1** — full implementation of the W3C query language
- **Full-text search** — Apache Lucene-powered indexing with facets and fields
- **Application platform** — build and deploy complete web applications as XAR packages
- **REST and WebDAV** — standard interfaces for integration
- **Fine-grained security** — Unix-style permissions on collections and resources
- **ACID transactions** — crash-safe with write-ahead logging

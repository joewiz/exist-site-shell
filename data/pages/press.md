# Press

## About eXist-db

eXist-db is an open source native XML database and application platform. It combines a high-performance, schema-less database with a complete application development framework — all built on XML technology and the XQuery language.

## Key Facts

- **First release:** 2001
- **License:** LGPL 2.1 (open source)
- **Language:** Java (database engine), XQuery (applications)
- **Standards:** XQuery 3.1, XPath 3.1, XSLT, REST, WebDAV
- **Package ecosystem:** 50+ community packages available
